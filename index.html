Projects

Robot Grasping Simulation


Project Description

This project was performed as part of the semi autonomous care robot project at Heemstech Innovative Technology (HiT) in the TU Delft Robo Valley. The goal was to develop a simulation that could be used to train the robot using machine learning/AI to autonomously grasp object. The simulation was the first step towards an AI driven robot. A colleague from Avans was simultaneously working on creating a grasping algorithm that was ultimately used to validate this simulation.

Project Technologies

The simulation was created for the Kinova Mico robotic arm. This arm was integrated with the iteration of the care robot Rose. The robot used the Robot Operating System (ROS) and therefore a key requirements was that the simulation integrated with this as well. Gazebo was used for this.

Project Outcomes

The main outcome was the simulation that you can see on the picture above. The arm was able to grasp the bottle with a similar accuracy as measured in real life. The behavior of the robot was similar with only minor differences in the fingertip actuation as the physical robot used under-actuated finger. This was not possible to be simulated due to time constraints.
